My Projects

A showcase of games and such I've developed

Project: War Ant

Role: Lead Developer | Engine: Unity | Year: 2024-Present

A personal project made with just me and a friend. It's a multiplayer turn-based RPG where you make a team of different classes of ants and fight other players. It's made using Unity and C#.

  • Unity
  • C#

Feathered Peril

Role: Game Designer & Developer | Engine: Unity | Year: 2025

A game made to be played by a large audience in a planetarium. It focuses on teaching players about birds and their troubles. The players connect to the game to interact with it using their phone (possible due to AirConsole integration). It was accepted to be presented at the Serious Games Conference in 2025.

  • Unity
  • C#
  • HTML
  • AirConsole

Michigan State University Internship Games

Role: Game Designer & Developer | Engine: Unity | Year: 2025

A collection of the two projects currently done from my game development internship work at MSU. The first is a resource management game meant to teach how to run a business, and this game is now used in a MSU class. The second is a choose your path kind of game to make finding resources easier for those who need to find them from the business college.

  • Unity
  • C#

Class Projects Collection

Role: Game Designer and Developer | Various Engines | Year: 2022-2025

Some notable projects I've done for some MSU classes. The first a partial recreation of Mega Man 2's Woodsman Stage in Unity. The second an Unreal grappling parkour game where you must swing to the end without hitting the ground and doing so fast enough to avoid running out of time. The third a stealth platformer game where you must get to the end without being spotted which was also made in Unreal. Finally is a 3D FPS game made in Unity where you must find and defeat all the enemies.

  • Unity
  • C#
  • Unreal Engine
  • Visual Scripting
